Задача:
Преобразуй переданный текст в корректный Markdown со строгой структурой.

Структура:
- Первый уровень — заголовок H1.
- Разделы — заголовки H2.
- Все пункты внутри разделов ОБЯЗАТЕЛЬНО оформить маркированным списком "-".
- Не должно быть строк вне списков внутри разделов.

Строгие правила:
- Нельзя менять текст.
- Нельзя добавлять информацию.
- Нельзя удалять информацию.
- Можно только форматировать Markdown.

Формат ответа:
Верни только готовый Markdown без пояснений.
